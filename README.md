# Vacation Rating App
A school project for JOHN BRYCE. This app allows you to create a vacation card (containing: vacation name, picture, price and rating)

## Key Features
* create a vacation card containing: vacation name, picture, price and rating by filling a form.
* 1-4 stars icon vacation rating display.
* Adding likes possibility.
* Vacations cards are saved to Local Storage of the browser. When returning to the page after closing it the vacations carda will still appear on the task board.
* Vacation can be easily deleted

## Tech Stack
* Frontend: HTML5, Bootstrap5
* Logic: Vanilla JavaScript (ES6+)

## How to use
* Fill all the form's input fields.
* Don't forget to rate your vacation in the "Rate it" drop down list.
* Click the 'Create a vacation card' button
* Your vacation card will appear.
* You can add likes by clicking the :+1: button on the vacation card. Number of likes will appear beside it.
* To delete a vacatin card simply click the "X" button that appear in its top-right corner
